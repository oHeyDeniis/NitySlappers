# NitySlapper

Slappers functions in serenity!

## Features

- Human Slappers
- Commands

## Comming Features

- Mobs entities
- form control
- more modifications for devs

## Usage
- /slapper create [name] [id] - Creates a slapper with the given name and id.
- /slapper remove [id] - Removes the slapper with the given id.
- /slapper list - Lists all slappers in the current dimension.
- /slapper addcmd [id] [cmd] - Adds a command to the slapper with the given id. (use . for spaces)
- /slapper removecmd [id] [cmd] - Removes a command from the slapper with the given id. (use . for spaces)
- /slapper tpme [id] - Teleports the slapper with the given id to you.
- /slapper tp [id] - Teleports you to the slapper with the given id.
- /slapper setname [id] [new.name] - Sets the name of the slapper with the given id. (use . for spaces)
- /slapper chunklist - Lists all entities in the current chunk.
